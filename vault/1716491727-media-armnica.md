- La media armónica de una serie de observaciones $\{x_1,x_2,...,x_n\}$ se utiliza para promediar rendimientos, velocidades... variables que se expresan en términos relativos
$$H=\frac{n}{\sum_{i=1}^n \frac{1}{x_i}}$$
