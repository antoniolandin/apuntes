# Error relativo

El [[1707819341-errores|error]] relativo es la relación entre el [[1707819574-error-absoluto|error absoluto]] y el valor exacto. Se calcula con la fórmula:

$${E = \frac{|N-V|}{|V|}}$$

$V$ es el valor exacto y $N$ es el valor aproximado.
