# Existe un único inverso multiplicativo módulo primo

## Enunciado

Si $p$ es un número primo y $a$ es un número entero tal que $0 < a < p$, entonces existe un único entero $b$ tal que $0 < b < p$ y $ab \equiv 1 \pmod{p}$.
