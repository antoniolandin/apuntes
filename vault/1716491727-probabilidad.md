1. $P(\emptyset) = 0$
2. $P(A^c)=1-P(A)$
3. $P(U_{i=1}^n A_i), \; \text{ si } (A_i \cap A_j) = \emptyset, \; \forall i \neq j$

5. $P(A \cup B) = P(A) + P(B) - P(A \cap B)$
6. $P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(A \cap B) - P(A \cap C) + P(A \cap B \cap C)$
