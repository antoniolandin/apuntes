# Error absoluto

El [[1707819341-errores|error]] absoluto es la diferencia entre el valor exacto y el valor aproximado. Se calcula con la fórmula:

$${E = |N-V|}$$

$V$ es el valor exacto y $N$ es el valor aproximado.
