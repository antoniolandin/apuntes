# Matriz traspuesta

La **matriz traspuesta** de una [[1708971486-matriz|matriz]] $A$ de orden $m \times n$ se obtiene intercambiando filas por columnas. Se denota por $A^t$.

$${(a_{ij})' = a_{ji}}$$

## Propiedades

1. ${(A^t)^t = A}$
2. ${(A + B)^t = A^t + B^t}$
3. ${(kA)^t = kA^t}$
4. ${(AB)^t = B^tA^t}$
